在手机上通过termux实现vtb机翻

2020-03-26

youtube-streaming-translator-python
我这里的环境termux aarch64 python3.8
大体上来说，报错可以直接下离线包,我这里只是我的情况
请自行随机应变
离线包记得改对应的名字
首先下载termux
还是用魔改的uTermux吧，原版太麻烦了
左划换清华源，一直y就对了，大概

pkg install python
pip install --upgrade pip
git clone https://github.com/azuse/youtube-streaming-translator-python
cd https://github.com/azuse/youtube-streaming-translator-python
pip install -r requiremments.txt

这里安装opencv-python报错了，我选择了安装离线包
自己下吧https://pypi.tuna.tsinghua.edu.cn/simple/opencv-python/

pip install opencv_python-4.2.0.32-cp38-none-linux_aarch64.whl
pip install -r requiremments.txt

下面不重复了，解决错误后继续

pip install -r requiremments.txt

安装scipy又报错了

pkg install wget
$PREFIX/bin/wget https://its-pointless.github.io/setup-pointless-repo.sh

apt install scipy

安装grpcio又报错了,依旧是离线包
https://pypi.tuna.tsinghua.edu.cn/simple/grpcio/

pip install grpcio-1.28.0rc3-cp38-none-linux_aarch64.whl

pkg install portaudio
